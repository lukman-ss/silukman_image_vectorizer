# Performance Guideline

Last updated: 2026-06-05

## Fastest Inputs

- Flat logos.
- Icons.
- Badges.
- Stickers.
- Clean character art.

## Heavier Inputs

- Photos.
- Complex gradients.
- High-resolution artwork with many colors.
- Images with shadows, blur, texture, or noise.

## Tips

- Use `Logo` or `Icon` presets for simple 2D artwork.
- Use `Photo` only when high visual similarity is more important than SVG size.
- Increase `Min Area` to remove tiny shapes.
- Lower custom color count to reduce file size.
- Use palette replacement on clean color regions.
- Remove shadows from benchmark artwork when testing flat vector output.
