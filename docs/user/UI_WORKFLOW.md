# UI Workflow

Last updated: 2026-06-05

## Basic Workflow

1. Click `Import Image`.
2. Review original image and metadata.
3. Choose a quality preset.
4. Tune hand-pick settings if needed.
5. Review vectorized result.
6. Click `Export SVG`.

## Palette Editing Workflow

1. Import an image.
2. Review `Input Color Palette`.
3. Click a swatch to replace a dominant color.
4. Or click `Pick Color From Image`.
5. Click a pixel in the original preview.
6. Choose a replacement color.
7. Wait for vectorization to update.
8. Export SVG.

## Batch Workflow

1. Click `Select Batch Images`.
2. Review valid and invalid files.
3. Click `Process Batch`.
4. Select output folder.
5. Wait for progress and final summary.
