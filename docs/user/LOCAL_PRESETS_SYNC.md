# Local Presets Sync

Last updated: 2026-06-05

Preset choices are currently local UI settings inside the running application.
There is no external sync service.

## Current Presets

- Logo.
- Artwork.
- Icon.
- Photo.
- Custom.

## Stored Preferences

Theme mode is persisted with `QSettings`.

## Not Synced

- Current imported image.
- Current vector result.
- Palette replacement mappings.
- Batch file selection.
