# Project Overview

Last updated: 2026-06-05

Image Vectorizer is a desktop application for converting raster images into SVG
vector output with a PySide6 UI.

## Core Value

- Local raster-to-vector conversion.
- Visual side-by-side comparison.
- Quality presets for logo, icon, artwork, and photo workflows.
- Direct SVG export.
- Batch SVG export.
- Color palette editing from image-detected or user-picked source colors.

## Target Inputs

- PNG
- JPG/JPEG
- BMP
- WEBP

## Target Output

- SVG

## Primary Users

- Designers preparing SVG assets.
- Developers converting UI icons or simple illustrations.
- Users who need local conversion without cloud processing.
