# Roadmap

Last updated: 2026-06-05

This roadmap captures possible future work. It does not override the current
implementation status.

## Short-Term

- Improve palette editing UX with selected-source highlighting.
- Add replacement tolerance control.
- Add inline SVG complexity warnings.
- Add benchmark command script for repeatable sample metrics.

## Mid-Term

- Add undo/redo for palette edits.
- Add preset export/import for vector settings.
- Add visual color usage percentages.
- Add more sample datasets for icons and mascot art.

## Long-Term

- Add optional SVG optimization pass.
- Add more packaging targets and release automation.
- Add richer in-app help documentation.
